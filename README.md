# 📊 Pandas Data Analysis Project

## 📌 Project Overview
This project demonstrates data analysis using Python pandas. It includes data cleaning, processing, and extracting useful insights from the dataset.

---

## 📊 Dataset Information
The dataset contains employee records with the following columns:
- Name
- Age
- City
- Salary
- Department

---

## ⚙️ Tasks Performed
- Data cleaning
- Handling missing values
- Removing duplicates
- Filtering data
- GroupBy analysis
- Sorting and ranking

---

## 📈 Key Insights
- Average salary by city
- Highest salary by department
- Employee distribution by location

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- Jupyter Notebook

---

## 🚀 Project Goal
To practice real-world data analysis skills and build a strong portfolio project.
